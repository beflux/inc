# デザインコンセプト

## 概要

Beflux, Inc. のコーポレートサイトは、**ミニマリズム**と**静謐さ**を基調としたデザインです。
余計な装飾を排し、本質的な情報のみを提示することで、信頼感とプロフェッショナリズムを表現します。

## デザイン原則

### 1. Less is More
- 必要最小限の要素のみを配置
- 余白を活かしたレイアウト
- 視覚的なノイズを排除

### 2. Typography First
- タイポグラフィを主役としたデザイン
- 軽やかなフォントウェイト（300）で洗練された印象
- 適度なレタースペーシングで可読性を確保

### 3. Subtle Interaction
- 控えめなアニメーションでユーザーを迎える
- ホバー時の色変化など、さりげないフィードバック
- 派手な演出よりも上質な体験を重視

## カラーパレット

| 用途 | カラー | 説明 |
|------|--------|------|
| 背景 | `#ffffff` | 純白でクリーンな印象 |
| メインテキスト | `#2d2d2d` | 黒すぎない、柔らかいダークグレー |
| サブテキスト | `#707070` | 控えめなミディアムグレー |
| テキスト（base） | `#1a1a1a` | ベースカラー |

## タイポグラフィ

### フォントファミリー
```css
"Inter", "Helvetica Neue", Arial, "Hiragino Kaku Gothic ProN", "Hiragino Sans", Meiryo, sans-serif
```

欧文はInter、和文はヒラギノ角ゴシックを基本とし、
どのデバイスでも美しく表示されるフォールバックを設定。

### フォントサイズ
- **会社名**: `clamp(2rem, 8vw, 4.5rem)` - レスポンシブに可変
- **連絡先**: `clamp(0.875rem, 2vw, 1rem)` - 控えめなサイズ

### フォントウェイト
- 全体を通して **300（Light）** を使用
- 軽やかで現代的な印象を与える

## アニメーション

### フェードイン
```css
@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

- **会社名**: 1.2秒かけてフェードイン
- **連絡先**: 0.3秒遅延後にフェードイン（順番に表示される演出）

### ホバーエフェクト
- 連絡先リンク: `#707070` → `#2d2d2d` へ0.2秒かけて変化
- 下線なし、色の変化のみで上品に

## レイアウト

- **中央配置**: 画面中央に要素を配置
- **縦並び**: 会社名と連絡先を縦に配置
- **間隔**: 2rem のギャップで適度な余白を確保

## 今後の拡張指針

サイトを拡張する際も、以下の原則を維持すること：

1. **追加する要素は最小限に** - 本当に必要な情報のみ
2. **同じトーンを維持** - 軽いフォントウェイト、控えめな色使い
3. **アニメーションは控えめに** - フェードイン程度に留める
4. **余白を大切に** - 詰め込みすぎない
